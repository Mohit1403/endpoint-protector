Autopentrix Security Platform

Enterprise Penetration Testing & Endpoint Protection Suite

1. Overview

Autopentrix is an integrated cybersecurity platform designed to automate penetration testing, threat intelligence, and endpoint monitoring. It enables security teams to conduct assessments, analyze risks, and manage endpoints through a centralized interface.

2. Architecture

  • Frontend Dashboard (Web UI)
  • Backend Server (Node.js + Express + Socket.IO)
  • Endpoint Agents (Python-based telemetry collectors)

Communication is handled through REST APIs and WebSockets for real-time updates.

3. Core Capabilities

  • Automated network scanning using Nmap
  • VirusTotal integration for threat intelligence
  • Real-time endpoint telemetry monitoring
  • Remote command execution on endpoints
  • Cryptographic utilities (AES, DES, 3DES)
  • Automated report generation

4. API Endpoints

  • POST /api/virustotal/file
  • POST /api/virustotal/url
  • POST /api/virustotal/hash
  • GET /api/endpoint-protector/agents
  • POST /api/endpoint-protector/agents/:id/command

5. Deployment

  • Hosted on Render as a Web Service
  • Persistent storage enabled for reports and logs
  • Environment variables configured securely

Health Check: /health

6. Endpoint Agent

The endpoint agent collects telemetry and executes commands such as:

  • Process monitoring
  • USB control
  • Network interface control
  • User management

7. Security Notice

This platform is intended strictly for authorized security testing and controlled environments. Unauthorized use is prohibited.

8. Future Roadmap
